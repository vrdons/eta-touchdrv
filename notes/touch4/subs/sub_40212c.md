# sub_40212c

## Control flow

USB control-read helper. Initializes local 0x3a byte transfer buffer and calls lower USB helper with command `0xae`, request type `0x1e`, requested id byte, and max receive length `0x3a`. If transfer succeeds and returned length is `<= 8`, copies that many bytes into caller output and writes length byte to caller length pointer. Returns `1` on success, otherwise `0`.

Called by [[sub_4027b4]].

```c
int sub_40212c(void *dev, unsigned char id, unsigned char *out, unsigned char *out_len)
{
    unsigned char buf[0x3a];
    uint64_t got = 0;
    unsigned char i;

    if (sub_401e6b(dev, 0x1e, 0xae, id, buf, 0, 0, 0x3a, &got, 1) == 0)
        return 0;
    if (got > 8)
        return 0;

    for (i = 0; i < got; i++)
        out[i] = buf[i];
    *out_len = (unsigned char)got;
    return 1;
}
```
