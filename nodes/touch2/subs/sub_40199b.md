# sub_40199b

## Control flow

Debug symbol name: `OptClickFilterInitialize`. `filter == NULL` ise `0` döner. Null değilse internal `pro_ClickFilterInit(filter)` çağırır ve `-1` döner. [[sub_405d74]] init yolunda bu dönüşü truthy başarı olarak kullanır.

```c
int sub_40199b(void *filter)
{
    if (filter == 0)
        return 0;
    pro_ClickFilterInit(filter);
    return -1;
}
```
